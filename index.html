Kisah Perjalanan Saya

Saya adalah lulusan SMK Negeri 1 Juwangi - Boyolali yang memulai karir di industri manufaktur selama 7 tahun sebagai Operator Produksi di PT. Nippon Indosari Corpindo Tbk (2015-2022).

Ketertarikan saya pada teknologi membawa saya melanjutkan pendidikan ke Universitas Widya Husada Semarang, Program Studi S1 Sistem & Teknologi Informasi. Di semester 3, saya memutuskan untuk aktif berorganisasi dan bergabung dengan Dewan Perwakilan Mahasiswa (DPM) sebagai bentuk kontribusi nyata dalam kemahasiswaan.

Sebagai Anggota Komisi 2 DPM periode 2023/2024, saya tidak hanya aktif dalam advokasi mahasiswa, tetapi juga mengaplikasikan skill teknologi saya dengan membangun website resmi DPM dan meraih Juara 2 Lomba Web Design.

"Bergabung dengan DPM di semester 3 adalah komitmen saya untuk berkontribusi dalam kemahasiswaan sambil mengembangkan passion di bidang web development."
